Free Sales Leads API Documentation

Welcome to the API documentation for the Free Sales Leads API. This API provides access to curated leads data for real estate agents and YouTube creators.

For any inquiries or assistance, please contact us at contact@collinsawusa.com.

Endpoints

Real Estate Agents Leads

Get a list of leads for real estate agents.

  • Endpoint: /api/real-estate-agents-leads/
  • HTTP Method: GET
  • Response: List of real estate agent leads

Request

No request parameters required.

Response

Status Code: 200 (OK)

[
  {
    "id": "1",
    "business_name": "Example Realty",
    "phone": "123-456-7890",
    "website": "http://www.example.com",
    "address": "123 Main St, City",
    "created_at": "2023-08-14T12:00:00Z",
    "modified_at": "2023-08-14T12:30:00Z"
  },
  // ... more leads
]

YouTube Creators Leads

Get a list of leads for YouTube creators.

  • Endpoint: /api/youtube-creators-leads/
  • HTTP Method: GET
  • Response: List of YouTube creator leads

Request

No request parameters required.

Response

Status Code: 200 (OK)
